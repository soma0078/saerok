# 새록 (Saerok)

저장한 문장(말씀, 명언, 다짐 등)을 카테고리별로 관리하고, 설정한 시간에 로컬 알림으로 반복 상기시켜주는 iOS/Android 앱.

## 기술스택

| 분류 | 기술 |
|------|------|
| 프레임워크 | Expo (React Native, TypeScript) |
| 스타일링 | Uniwind (Tailwind CSS v4 for React Native) |
| 데이터 저장 | AsyncStorage (로컬, 서버 없음) |
| 알림 | expo-notifications |
| 배포 | App Store / Google Play |

## 시작하기

```bash
npm install
npm start          # Expo 개발 서버
npm run ios        # iOS 시뮬레이터
npm run android    # Android 에뮬레이터
```
